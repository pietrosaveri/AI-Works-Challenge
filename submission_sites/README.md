# Portfolio Submission Sites

## Overview
This folder contains three professionally generated portfolio websites for submission to judges. Each site is a unique, AI-generated "anti-portfolio" that showcases authentic personality through thoughtful questionnaire responses.

## Personas

### Person 1: Maya Rodriguez
**Environmental Scientist**
- **Location:** `person1_linus/`
- **Profile:** An environmental scientist passionate about sustainable living and community engagement
- **Design Theme:** Ocean Teal with Dolphin spirit - representing harmony and collaboration
- **Highlights:** Focuses on solving complex environmental challenges through community collaboration

### Person 2: James Chen
**Software Engineer**
- **Location:** `person2_steve/`
- **Profile:** A software engineer obsessed with elegant code, vintage keyboards, and custom audio equipment
- **Design Theme:** Midnight Purple with Owl spirit - representing recursion and deep thought
- **Highlights:** Values clean code and learned that done is better than perfect

### Person 3: Sofia Andersson
**Community Organizer & Facilitator**
- **Location:** `person3_douglas/`
- **Profile:** A connector of people who creates spaces for vulnerability and innovation
- **Design Theme:** Warm Amber with Golden Retriever spirit - representing synergy and warmth
- **Highlights:** Hosts dinner parties where strangers become friends, believes in intentional community

## Structure

Each person's folder contains:
```
personX_name/
├── index.html          # Main portfolio page
└── assets/             # Stylesheets, fonts, and resources
```

Additionally, each person has a corresponding JSON file documenting their questionnaire responses:
- `person1_linus.json` - Maya Rodriguez's profile data
- `person2_steve.json` - James Chen's profile data
- `person3_douglas.json` - Sofia Andersson's profile data

## Technical Details

- **Generation Method:** Multi-agent AI system with specialized agents for mood/design, content strategy, UX architecture, icon curation, and React development
- **Design Approach:** Each site features a unique color palette, typography, and layout style tailored to the individual's personality
- **Content:** Fully dynamic content generated from questionnaire responses, creating authentic personal narratives

## Viewing the Sites

To view any site, simply open the respective `index.html` file in a web browser:
- `person1_linus/index.html` - Maya Rodriguez's portfolio
- `person2_steve/index.html` - James Chen's portfolio
- `person3_douglas/index.html` - Sofia Andersson's portfolio

All sites are self-contained and require no build process or server to run.

## Design Philosophy

Each site embodies an "anti-portfolio" approach:
- **Authenticity over perfection** - Real struggles and lessons learned
- **Personality-driven design** - Color, animal, and abstract word influence visual identity
- **Thoughtful content** - Deep reflection on mistakes, values, and what makes each person unique beyond their job title
